# Ran Online – Modern Website Design Skill

This skill guides the creation of distinctive, production-grade website interfaces for **Ran Online**.

The new website architecture is **hybrid**:

1. A cinematic, marketing-focused public site
2. A lightweight but immersive account dashboard

Both must feel unified under one strong urban combat identity.

Typography, spacing scale, design tokens, and theme variables are already defined in `globals.css`.

Your responsibility is to execute a bold, intentional aesthetic direction aligned with Ran Online’s world.

---

# Core Identity

Ran Online is:

- Urban fantasy
- School-based faction rivalry
- Martial arts combat
- PvP-heavy progression
- Competitive culture
- Neon city atmosphere
- Hardcore grind ecosystem

The website must feel like:

> A digital command center for urban warriors.

Not:

- A SaaS product
- A startup landing page
- A nostalgic early-2000s MMO relic
- A generic gaming template

---

# Hybrid Architecture Rule

The website consists of two layout systems:

## 1️⃣ Marketing Landing Layout (Public)

Purpose:

- Acquire players
- Reinforce identity
- Present expansions
- Highlight faction rivalry
- Showcase PvP dominance

Tone:
Cinematic. Aggressive. Immersive.

This layout is:

- Visually bold
- Atmosphere-driven
- Asymmetrical
- Emotion-heavy

It should feel like:
A recruitment broadcast from a rival academy.

---

## 2️⃣ Dashboard Layout (Logged-In)

Purpose:

- Account management
- Character overview
- Rankings
- Guild data
- Progress tracking
- PvP statistics

Tone:
Tactical. Sharp. Controlled.

This layout is:

- Structured but not SaaS-like
- High contrast
- Prestige-focused
- Minimal but powerful

It should feel like:
A command terminal inside an elite academy.

---

# Critical Rule

The dashboard must not feel detached from the marketing site.

Both layouts must share:

- Typography system
- Faction color dominance
- Urban texture language
- Visual sharpness
- Competitive tone

The marketing site sells the war.
The dashboard manages your position in it.

---

# Design Philosophy

## Neo-Urban Combat Minimalism

Visual language:

- Dark asphalt / concrete base layers
- Steel undertones
- Subtle grain or noise overlays
- Tactical line separators
- Controlled neon accent usage
- Sharp, intentional spacing
- Large typographic hierarchy

Think:

Tokyo underground fight club
Meets supernatural academy warfare.

---

# Marketing Layout Guidelines

## Hero Sections

Must include:

- Massive display typography
- Strong faction-driven visual moment
- Asymmetry
- Depth layering
- Clear CTA hierarchy
- One signature visual mechanic

Avoid:

- Centered generic hero
- Equal 3-column layouts
- Template-style sections
- Overuse of glow effects

Use:

- Diagonal splits
- Overlapping typography
- Cinematic section transitions
- Scroll-based reveal pacing

---

## Expansion / Event Pages

- Large section headers
- Strong visual breaks
- Prestige framing
- Impact highlights
- Clean but aggressive typography

Avoid blog-style layouts.

---

# Dashboard Layout Guidelines

The dashboard must avoid SaaS admin UI patterns.

Do NOT:

- Use generic card grids
- Use soft rounded UI everywhere
- Use evenly spaced panels
- Create analytics-style layouts

Instead:

- Use sharp framed panels
- Large numeric dominance
- Rank insignia emphasis
- Character portrait as focal anchor
- Subtle faction watermark background
- Tactical spacing rhythm

Panels should feel forged, not templated.

---

# Faction System Support

Faction identity must be theme-aware:

Sacred Gate → Gold dominance
Mystic Peak → Blue dominance
Phoenix → Red dominance

Rules:

- Only one faction color dominates at a time
- Accent used strategically
- No rainbow mixing
- Red reserved carefully for alerts and high-intensity signals

Dashboard theme should dynamically adjust based on faction.

---

# Typography Strategy

Typography is already defined in `globals.css`.

Use it intentionally:

- Massive display headers for marketing
- Condensed uppercase for rankings
- Tight tracking for authority
- Large numeric stats for dominance
- Controlled body copy blocks

Text should feel like:
Combat declarations, not marketing filler.

---

# Motion Strategy

Marketing Site:

- Staggered hero reveal
- Section fade-in on scroll
- Subtle parallax depth
- CTA glow sweep
- Controlled glitch or distortion effects

Dashboard:

- Minimal animation
- Hover elevation
- Rank highlight transitions
- Subtle numeric change animations

One strong animated moment per page is required.

Avoid:

- Continuous particle spam
- Playful bounce
- Excessive blur
- Over-animated UI chrome

---

# Layout & Composition

Avoid:

- Symmetrical marketing stacking
- Bootstrap-style cards
- SaaS dashboard templates
- Safe predictable grids

Use:

- Asymmetry
- Offset compositions
- Edge-to-edge sections
- Diagonal dividers
- Large typography breaking grid
- Controlled density in dashboard

Composition should feel slightly aggressive.

---

# Atmospheric Layer Requirement

Each page must include at least one:

- Subtle grain overlay
- Faction crest watermark
- Neon edge highlight
- Tactical border framing
- Urban texture background
- Scanline or digital layer

Never use flat empty backgrounds unless stylistically justified.

---

# Signature Visual Mechanic (Mandatory)

Each major page must include one defining visual moment:

Examples:

- Headline glitch distortion
- Animated faction border lines
- Neon underline sweep
- Diagonal energy split
- Rank badge illumination on scroll
- Large numeric fade-in impact

Without this, the design lacks memorability.

---

# Anti-Patterns (Strictly Forbidden)

Do NOT:

- Recreate early 2000s MMO UI
- Use beveled glossy buttons
- Use centered template heroes
- Use evenly spaced marketing blocks
- Create SaaS-style dashboards
- Overuse drop shadows
- Use “gamer purple gradient” aesthetics
- Overcrowd with visual noise

Modern evolution only.

---

# Implementation Standards

- Production-grade code
- Modular layout separation (MarketingLayout / DashboardLayout)
- CSS-first animation preferred
- Optimized hero media
- Accessible contrast
- Performance-aware
- Clean semantic structure
- Responsive while preserving strong identity

---

# Final Rule

This website is not just promotional.

It is:

- A rivalry arena
- A prestige leaderboard
- A recruitment weapon
- A PvP command system
- A digital academy battleground

If it feels corporate or safe, redesign it.

Design like it belongs inside the war.
Not exported from a template.

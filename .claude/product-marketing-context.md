# Product Marketing Context

## Product Overview

Product Name: Ran Online GS
Type: Private MMORPG Game Server
Genre: Online Multiplayer RPG (MMORPG)
Platform: PC, Desktop
Region: Global (primarily Southeast Asia, Philippines, and Latin America)

Ran Online GS is a private server version of the classic MMORPG Ran Online. It provides custom gameplay, balanced PvP, unique events, and an active community for players who want a modernized version of the original game.

The server focuses on competitive gameplay, school-based factions, and fast progression systems that appeal to both nostalgic players and new MMORPG fans.

## Technical SEO for Keywords

### URL Structure
- `/` - Homepage (primary brand + generic keywords)
- `/download` - Download page
- `/register` - Registration
- `/features` - Server features
- `/news` - News/updates (future)
- `/guides` - Guide hub (future)
- `/guides/[slug]` - Individual guides (future)
- `/events` - Events calendar (future)

### Internal Linking Strategy
```
Homepage
  ├── Download (CTA from hero)
  ├── Register (CTA from hero)
  ├── Features (nav link)
  ├── News (nav link)
  └── Guides (nav link)
      ├── Beginner Guide
      ├── PvP Builds
      └── Farming Spots

Download → Register (cross-CTA)
Features → Download (cross-CTA)
All pages → Homepage (logo link)
```

### Anchor Text Guidelines
- Use keyword-rich anchor text for internal links
- Vary anchor text (don't always use exact match)
- Examples:
  - "Download Ran Online GS" (to download page)
  - "Create your free account" (to register page)
  - "See all server features" (to features page)
  - "Read our beginner guide" (to guide)
